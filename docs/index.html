Based on JSON Schema

JSON BinPack adopt the industry-standard schema language for JSON used by OpenAPI, AsyncAPI, RAML and more. No extensions or custom vocabularies. JSON BinPack encodes any JSON document given a matching JSON Schema definition.

Schema-driven and Schema-less

JSON BinPack is a hybrid format that supports both schema-driven and schema-less modes. It allows you to be as flexible or as strict as you want to be.

Highly Space-efficient

JSON BinPack has been found to be more space-efficient than ASN.1, Apache Avro, Microsoft Bond, Cap'n Proto, FlatBuffers, Protocol Buffers, Apache Thrift, BSON, CBOR, FlexBuffers, MessagePack, Smile and UBJSON for every case of a JSON dataset.

Industry Meets Research

JSON BinPack has been designed as part of a research project at University of Oxford.
